# Certificate Generator

Static HTML/CSS/JS certificate generator based on `12.pptx`.

## Files

- `index.html` - app page
- `styles.css` - layout, print styles, certificate typography
- `script.js` - live text updates and PNG export
- `assets/` - images extracted from the PowerPoint template

## GitHub Pages

1. Upload this folder to a GitHub repository.
2. In GitHub, open `Settings` > `Pages`.
3. Set the source to the branch and folder containing `index.html`.
4. Open the published Pages URL.

The page runs fully in the browser. No server code or database is needed.
